# IPZ_lab1

# Статичний сайт-візитка

Створено для лабораторної роботи з дисципліни «Інженерія програмного забезпечення».

**Використані інструменти:**
- Docker з Ubuntu 24.04
- Gemini CLI (встановлено та авторизовано, але інтерактивний режим мав проблеми з вводом — відома issue в контейнерах)
- GitHub + GitHub Pages

**Як запустити на GitHub Pages:**
1. Репозиторій публічний.
2. Settings → Pages → Source: Deploy from a branch → main → / (root) → Save.

Сайт доступний за адресою: https://твій_нік.github.io/назва-репозиторію/
