# IPZ_lab1 — Сайт-візитка Мирослави

Простий, сучасний та адаптивний статичний сайт-візитка, створений для студентки НУ "Чернігівська політехніка" (спеціальність "Комп'ютерна інженерія") в межах лабораторної роботи з дисципліни «Інженерія програмного забезпечення».

## Використані інструменти
- **Docker** з Ubuntu 24.04
- **Gemini CLI**: Використано для генерації контенту та структури.
- **HTML5 & CSS3**: Семантична розмітка та адаптивна верстка.
- **GitHub Pages**: Для хостингу.

## Структура проєкту
- `index.html`: Головна сторінка сайту.
- `style.css`: Файл зі стилями.
- `README.md`: Опис проєкту та інструкції.

## Як розгорнути на GitHub Pages
1. Перейдіть до свого репозиторію на GitHub.
2. Відкрийте вкладку **Settings** (Налаштування).
3. У меню зліва оберіть пункт **Pages**.
4. У розділі **Build and deployment** переконайтеся, що в полі **Source** обрано `Deploy from a branch`.
5. У полі **Branch** оберіть `main` і папку `/ (root)`.
6. Натисніть **Save**.
7. Через кілька хвилин ваш сайт буде доступний за адресою: `https://3Mira3.github.io/IPZ_lab1/`
